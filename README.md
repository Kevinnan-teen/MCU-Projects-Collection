# 单片机项目合集

## 1、排队叫号系统设计

类似食堂取餐的模式。首先，学生排队，服务员使用键盘给顾客分配号码，按下确定键，数码管显示当前队列中的人数、刚刚进入队列的号码，并通过串口通信的方式将队列人数和当前分配的号码发送给从机，从机将数据显示在1602液晶屏上。另外，主机通过键盘按键选择取餐功能，按下出队键，叫到要取餐的号码，并通过语音播报取餐号码。
